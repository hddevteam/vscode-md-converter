Powerful Features

Everything you need to convert documents efficiently and accurately

Word Documents

Convert .docx and .doc files to Markdown while preserving formatting, headings, lists, and tables.

Excel Spreadsheets

Transform .xlsx, .xls, and .csv files into clean Markdown tables with automatic data formatting.

PDF Documents

Extract and optimize text from PDF files with advanced processing algorithms and smart formatting.

Batch Processing

Process entire folders of documents at once with intelligent file type detection and bulk operations.

Multi-Language

Automatic language detection with full English and Chinese interface support for global users.

Easy Integration

Right-click context menus and command palette integration for seamless workflow integration.

Get started in less than 30 seconds

1

Open VS Code Extensions

2

3

Install & Start Converting

How to Use

Multiple ways to convert your documents

Context Menu

Right-click on any supported document file in the VS Code explorer and select the appropriate conversion option.

Command Palette

Press Ctrl+Shift+P and type conversion commands like "Convert Word to Markdown".

Batch Processing

Right-click on folders to batch convert multiple documents with customizable options.
